● 초급 Getting Started Claude를 처음 써보는 단계 — 계정 가입과 Desktop 앱 설치
G-1 초급 claude.ai 회원가입부터 Claude Desktop 설치까지
김 국장은 이번 5일 과정을 시작하기 전, 아직 Claude 계정이 전혀 없는 상태입니다. 코딩 경험이 없어 무엇부터 눌러야 할지 막막하지만, 오늘은 딱 두 가지만 해내면 됩니다 — claude.ai 계정 만들기, 그리고 컴퓨터에 Claude Desktop 앱을 설치해 로그인까지 마치기입니다.
사용 요소
claude.ai 계정 가입 Claude Desktop 설치
단계별 실행
  1. 1. claude.ai 접속 및 로그인 방식 선택

    브라우저를 열고 claude.ai에 접속합니다. 화면 가운데에 있는 "Continue with email" 또는 "Continue with Google" 버튼 중 편한 쪽을 클릭합니다.

  2. 2. 이메일 인증 완료

    "Continue with email"을 선택했다면 입력한 주소로 로그인 링크가 담긴 메일이 도착합니다. 메일을 받은 것과 같은 기기라면 링크 클릭만으로 로그인되고, 다른 기기(예: 휴대폰으로 메일 확인)라면 화면에 표시된 인증 코드를 입력합니다.

  3. 3. 전화번호 및 나이 확인, SMS 코드 입력

    신규 계정이므로 국가번호(대한민국은 +82)와 전화번호를 입력하고, 만 18세 이상임을 확인하는 절차를 거칩니다. 이어서 문자로 오는 6자리 코드를 입력하면 가입이 끝납니다.

  4. 4. Claude Desktop 다운로드

    웹 브라우저에서 claude.com/download에 접속한 뒤, 사용 중인 컴퓨터가 Windows라면 "Download for Windows" 버튼을, macOS라면 "Download for macOS" 버튼을 클릭합니다.

  5. 5. 설치 및 로그인

    내려받은 설치파일을 더블클릭해 실행하고, 화면 안내를 그대로 따라가면 설치가 끝납니다. 설치 후 앱을 실행해 1~3단계에서 만든 계정으로 로그인합니다.

claude.ai 계정은 클라우드에 저장되는 하나의 로그인 정보이며, Claude Desktop·모바일 앱·Cursor의 Claude Code 확장까지 모두 이 계정 하나로 연결됩니다. 즉 오늘 만든 계정 하나가 5일 내내 사용할 "출입증" 역할을 합니다.
가입 과정에서 인증 메일이나 문자가 바로 오지 않는다면, 스팸함을 먼저 확인하고 1~2분 정도 기다린 뒤 재전송을 시도하세요. 기관 이메일 계정은 외부 메일 수신이 지연되거나 차단되는 경우가 있으므로, 필요하다면 개인 이메일로도 시도해볼 수 있습니다.
● 중급 Daily Workflow 코드를 다룰 실제 작업대 — Cursor 설치와 Claude Code 확장 연결
D-1 중급 Cursor 설치 + Extensions 뷰에서 Claude Code 확장 검색·설치·로그인
G-1을 마친 김 국장은 이제 claude.ai 계정과 Claude Desktop을 모두 갖춘 상태입니다. 하지만 앞으로 4일간 실제로 Python 코드를 작성하려면 대화창만 있는 도구로는 부족합니다. 이번에는 코드 편집이 가능한 Cursor를 설치하고, 그 안에 Claude Code 확장을 연결해 "AI에게 코드 작업을 직접 시킬 수 있는 상태"를 만들어 봅니다.
사용 요소
Cursor 에디터 Claude Code 확장
단계별 실행
  1. 1. Cursor 다운로드 및 설치

    cursor.com/download에 접속해 사용 중인 운영체제용 설치파일을 내려받고 실행합니다. 설치가 끝나면 Cursor를 처음 실행할 때 나오는 기본 설정 화면(테마, 키보드 단축키 방식 등)은 기본값 그대로 두고 넘어가도 됩니다.

  2. 2. Extensions 뷰 열기

    Cursor 창에서 키보드 단축키 Ctrl+Shift+X(Windows)를 누르거나, macOS라면 Cmd+Shift+X를 누릅니다. 좌측에 네모 4개가 겹쳐진 모양의 아이콘을 클릭해도 같은 화면이 열립니다.

    Windows / Linux : Ctrl+Shift+X
macOS           : Cmd+Shift+X
  3. 3. "Claude Code" 검색 및 설치

    Extensions 뷰 상단 검색창에 Claude Code를 입력합니다. 검색 결과 중 게시자가 Anthropic으로 표시된 "Claude Code for VS Code" 항목을 찾아 Install 버튼을 클릭합니다.

  4. 4. 확장 실행 및 로그인 연결

    설치가 끝나면 좌측 사이드바에 Claude Code 아이콘이 새로 생깁니다. 이 아이콘을 클릭해 대화창을 연 뒤, 처음 실행 시 뜨는 로그인 안내를 따라 브라우저에서 claude.ai 계정으로 인증을 완료합니다.

Claude Code 확장은 별도의 API 키를 요구하지 않고, 브라우저 OAuth 인증만으로 계정의 Pro 플랜 이용 권한을 그대로 가져와 사용합니다. 즉 한 번 로그인해두면 Cursor를 다시 열 때마다 매번 로그인할 필요 없이 바로 대화창이 연결됩니다.
Free 플랜 계정으로 로그인을 시도하면 확장이 정상적으로 연결되지 않습니다. G-1에서 가입한 계정이 아직 Free 플랜이라면, day1-setup_Manual.html의 "2. claude.ai 계정 가입" 항목에 있는 Pro 플랜 업그레이드 절차를 먼저 완료하세요.
● 고급 Power User 첫 워밍업 실습 — Claude에게 실제 파일 작업을 맡겨 보기
P-1 고급 Claude Code에게 "5일 프로젝트 폴더 구조를 만들어줘"라고 요청하기
D-1까지 마친 김 국장은 claude.ai 계정, Claude Desktop, Cursor, Claude Code 확장을 모두 갖춘 상태입니다. 마지막으로 오늘 실습에서는 이번 5일 과정 전체를 이 확장 하나로 진행하게 된다는 것을 직접 체감해보기 위해, Claude Code에게 처음으로 실제 폴더와 파일을 만들어 달라고 요청해 봅니다. 이것이 5일 과정에서 Claude에게 처음으로 "말로 파일 작업을 시켜보는" 순간입니다.
사용 요소
Claude Code 확장 5일 로드맵
단계별 실행
  1. 1. 실습용 폴더를 Cursor에서 열기

    바탕화면 등 원하는 위치에 news-alert-bot이라는 빈 폴더를 하나 만들고, Cursor의 File 메뉴에서 Open Folder를 눌러 이 폴더를 엽니다. 이후 4일간 실습할 모든 코드가 이 폴더 안에 쌓이게 됩니다.

  2. 2. Claude Code 대화창에 요청 입력

    좌측 사이드바의 Claude Code 아이콘을 클릭해 대화창을 열고, 아래와 같이 이번 5일 과정에 맞는 폴더 구조를 만들어 달라고 자연스러운 문장으로 요청합니다. 명령어를 외울 필요 없이, 하고 싶은 말을 그대로 적으면 됩니다.

    지금부터 5일 동안 "내 이름이 들어간 뉴스 기사를 찾아
텔레그램으로 알려주는 봇"을 만드는 프로젝트를 진행할 거야.
이 폴더 안에 아래 내용을 정리한 README.md 파일 하나와,
앞으로 코드를 저장할 src 폴더를 만들어줘.
README.md에는 5일 계획(Day1 환경 준비, Day2 텔레그램 봇 만들기,
Day3 네이버 뉴스 API 연동, Day4 GitHub Actions 자동화 배포,
Day5 운영·예외처리)을 간단히 정리해줘.
  3. 3. 작업 내용 확인 및 승인

    Claude Code는 파일을 만들거나 수정하기 전에 어떤 파일을 어떻게 바꿀지 미리 보여주고 승인을 요청하는 경우가 많습니다. 제안된 내용(README.md 생성, src 폴더 생성)을 확인하고 이상이 없으면 승인(Approve 등 표시된 버튼 클릭)합니다.

  4. 4. 결과물을 Cursor 탐색기에서 직접 확인

    Cursor 좌측의 파일 탐색기(Explorer) 패널을 열어, 방금 요청한 README.md 파일과 src 폴더가 실제로 생성되었는지 눈으로 확인합니다. README.md를 클릭해 열어보고 5일 계획이 잘 정리되어 있는지도 확인합니다.

Claude Code는 대화창에 입력된 요청을 해석해, 실제로는 파일 생성·수정 같은 명령을 컴퓨터에 내리는 방식으로 동작합니다. 사람이 폴더를 만들고 마우스 우클릭으로 파일을 새로 만드는 작업을, 문장으로 요청하면 Claude가 대신 수행하는 것입니다. 다만 컴퓨터에 실제 변화를 일으키는 작업이므로, 승인 절차를 거치도록 설계되어 있습니다.
요청 문장이 구체적일수록(원하는 파일 이름, 폴더 이름, 들어갈 내용까지) Claude가 만드는 결과물이 기대에 가까워집니다. 이번 실습처럼 "무엇을, 어떤 이름으로, 어떤 내용으로" 만들어 달라고 적는 습관을 Day 2부터도 그대로 이어가면 좋습니다.
